(1) CSS INTRODUCTION :

CSS stands for Cascading Style Sheets and plays an important role in providing the styles to the HTML elements, which helps to make web pages presentable. 
CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.
it is used for styling on a web pages.
The standard file extension for CSS files is .css. When you create a CSS file, you typically save it with a .css extension, such as styles.css.

(3) CSS APPLY METHOD :

⇒ There are three ways of inserting a style sheet:
- (1) inline css - style attribute - (2) internal css - <style> ... </style> - (3) External css - with using <link>

(i). INLINE STYLE :

- Inline styles in HTML allow you to apply CSS styles directly to individual HTML elements using the style attribute.
Example :
HTML
Copy
<!DOCTYPE html> <html> <body> <p style="color: blue;">This is a paragraph with inline styles.</p> <p style="color:red;">This is a paragraph.</p> </body> </html>
​

Inline styles have the following characteristics:

Specificity: Inline styles have the highest specificity, meaning they override styles applied through external CSS files or internal <style> tags.
Scope: Inline styles apply only to the specific element they are applied to. They do not affect other elements on the page.
(ii). INTERNAL STYLE :
- Internal styles in HTML allow you to define CSS styles directly within the HTML document, using the <style> tag in the <head> section. This method allows you to keep your styles separate from your HTML content but still within the same file.
Example :
HTML
Copy
<!DOCTYPE html> <html> <head> <title>Internal Style Example</title> <style> body { background-color: #f0f0f0; } h1 { color: blue; } </style> </head> <body> <h1>This is a heading</h1> <p>This is a paragraph.</p> </body> </html>
​

Internal styles have the following characteristics:

Specificity: Internal styles have a higher specificity than external CSS files but lower than inline styles. They will override styles from external CSS files but be overridden by inline styles.
Scope: Internal styles apply to the entire document unless overridden by more specific selectors or inline styles.

(iii). EXTERNAL STYLE :

- External styles in HTML allow you to define CSS styles in a separate file and then link that file to your HTML document. This method keeps your styles separate from your HTML content, which can make your code more organized and easier to maintain.
(1) Create a CSS file (e.g., styles.css) with your styles:
CSS
Copy
/* styles.css */ body { background-color: #f0f0f0; } h1 { color: blue; }
​
(2). Link the CSS file to your HTML document using the <link> tag in the <head> section:
HTML
Copy
<!DOCTYPE html> <html> <head> <title>External Style Example</title> <link rel="stylesheet" type="text/css" href="styles.css"> </head> <body> <h1>This is a heading</h1> <p>This is a paragraph.</p> </body> </html>
​

External styles have several advantages:

Maintainability: Keeping styles in a separate file makes it easier to update and maintain them, especially in larger projects with multiple HTML files.
Reusability: External styles can be reused across multiple HTML files, which helps maintain consistency in your design.

(5) CSS BOX MODEL :

ALT
Content: The actual content of the element, such as text, images, or other media. The content area is where the element's content is displayed.
Padding: The space between the content and the element's border. Padding helps create space around the content, separating it from the border. Padding is transparent and does not have a background color by default.
Border: The border of the element, which surrounds the padding and content. Borders can have a specified width, style, and color. The border is drawn outside the padding but inside the margin of the element.
Margin: The space between the border of the element and adjacent elements. Margins create space between elements and help control the overall layout of the page. Like padding, margins are transparent and do not have a background color by default.

(6) CSS !IMPORTANT METHOD

- The !important keyword in CSS is used to give a style declaration higher priority than normal. When applied to a style rule, it overrides any other styles applied to the same element through other selectors
HTML
Copy

4. Using HSL values:

0 to 360
Hue: The hue value is represented as a degree on the color wheel, ranging from 0 to 360. It represents the base color. For example, 0 and 360 both represent red, 120 represents green, and 240 represents blue.
Saturation: The saturation value is represented as a percentage, ranging from 0% to 100%. It represents the intensity or purity of the hue. A saturation of 0% is a shade of gray (no color), while 100% is the full hue.
Lightness: The lightness value is also represented as a percentage, ranging from 0% to 100%. It represents the lightness or darkness of the color. A lightness of 0% is black, 50% is the normal color, and 100% is white.
CSS
Copy
p { color: hsl(120, 100%, 50%); /* HSL color (green) */ }

(11) CSS BORDER

the border property is used to define the border of an element. It can be used to set the width, style, and color of the border.

- orderd : top, right , bottom , left (1) first way :
Three main property’s are used :
(i) border style
(ii) border width
(iii) border color

CSS Border Style

The border-style property specifies what kind of border to display.
The following values are allowed:
dotted - Defines a dotted border
dashed - Defines a dashed border
solid - Defines a solid border
double - Defines a double border
groove - Defines a 3D grooved border. The effect depends on the border-color value
ridge - Defines a 3D ridged border. The effect depends on the border-color value
inset - Defines a 3D inset border. The effect depends on the border-color value
outset - Defines a 3D outset border. The effect depends on the border-color value
none - Defines no border
hidden - Defines a hidden border
example: border-style: solid;

CSS Border Width

The border-width property specifies the width of the four borders.
The width can be set as a specific size (in px, pt, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick:
example: border-width: 2px;

CSS Border Color

The border-color property is used to set the color of the four borders.
example: border-color: red;
